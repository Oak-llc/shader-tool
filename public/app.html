    Adapt for App
    Rewrites the shader in the editor to meet this tool's WebGL2 conventions — without changing the visual output.
    • Adds #version 300 es and precision highp float;
    • Injects u_time and u_resolution uniforms
    • Replaces gl_FragColor → fragColor, texture2D → texture
    • Converts varying/attribute to WebGL2 in/out
    • Adds // @param annotations so uniforms become sliders
    Useful when pasting shaders from Shadertoy or GLSL Sandbox.

    How to use ShaderTool

    Prompt panel left

    Type a description of the shader you want — be as specific or abstract as you like. Click Generate to send it to the AI.

    • Random prompt — rolls a random creative idea so you can explore without a starting point.
    • Enhance prompt — rewrites your description to be richer and more specific before generating.
    • Attach image — include a reference image and the AI will try to capture its look.
    • Sweeteners — quick-add style, technique, or color pills to steer the generation without rewriting your prompt.

    Viewport center

    Renders the fragment shader in real time using WebGL2. The canvas fills all available space and updates live as the shader runs.

    • Pause / Resume — freezes u_time so you can inspect a specific frame.
    • Reset Time — snaps time back to zero.
    • Fullscreen — expands the canvas to fill your screen.
    • Undo / Redo — steps through your shader history.

    GLSL Editor bottom

    A full Monaco editor (the same engine as VS Code) with GLSL syntax highlighting. You can paste, write, or edit shaders directly here.

    • Click Compile or enable Auto-compile in Settings to apply changes live.
    • Adapt for App — rewrites a pasted Shadertoy or GLSL Sandbox shader to meet WebGL2 conventions.
    • Enhance shader — asks the AI to improve the shader's visual quality or add a specific effect.
    • Switch to Compiler Logs in the dropdown to see errors and warnings from the last compile.

    Parameters right

    Every uniform with a // @param annotation becomes an interactive control here — sliders for numbers, color pickers for vec3, toggles for booleans.

    • Add parameter + — type a name and the AI will add the uniform, wire it into the shader, and annotate it automatically.
    • Edit / Delete — select parameters to remove them from the shader.
    • Save preset — capture the current slider state and recall it later from the presets bar at the top of the panel.

    Saved shaders left, bottom

    Every time you generate a new shader it's automatically saved. Click any entry in the Saved list to restore it — the shader, prompt, and parameters all come back.

    • Hover over an item to reveal the delete button.
    • Click + (top of the panel) to start a new session with a blank canvas.

    AI provider switcher top bar

    Use the provider buttons in the toolbar to switch between Anthropic, OpenAI, Gemini, and local models at any time. Add API keys for each service in Settings. Providers with a saved key are shown in full; others are dimmed.
